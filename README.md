Dengue Case Prediction and Data Analysis in Ceará
Sobre o projeto

Este projeto foi desenvolvido como parte de um trabalho acadêmico durante a faculdade, com o objetivo de aplicar conceitos de análise de dados e modelagem estatística em um problema real de saúde pública.

A proposta consiste em analisar dados de casos de dengue em diferentes municípios do estado do Ceará, buscando identificar padrões, compreender o comportamento da doença ao longo do tempo e, quando possível, realizar previsões com base nos dados disponíveis.

Objetivo

O principal objetivo do projeto é explorar dados epidemiológicos relacionados à dengue e aplicar técnicas de análise e regressão para:

Entender a distribuição dos casos entre municípios
Identificar possíveis tendências ou variações ao longo do tempo
Avaliar fatores que possam influenciar o aumento ou redução dos casos
Construir modelos simples de previsão
Metodologia

O projeto segue uma abordagem básica de análise de dados, incluindo:

Coleta e organização dos dados
Limpeza e tratamento das informações
Análise exploratória (EDA)
Aplicação de modelos de regressão
Interpretação dos resultados obtidos

As ferramentas utilizadas incluem principalmente Python e bibliotecas voltadas para análise de dados.

Contexto

A dengue é um problema recorrente em diversas regiões do Brasil, especialmente em áreas com clima favorável à proliferação do mosquito transmissor. A análise de dados nesse contexto pode contribuir para uma melhor compreensão do problema e servir como base para tomadas de decisão mais informadas.

Este trabalho tem caráter acadêmico e exploratório, com foco no aprendizado e na aplicação prática de conceitos estudados ao longo do curso.

Considerações finais

Este projeto representa uma aplicação inicial de técnicas de análise de dados em um cenário real. Apesar de suas limitações, ele demonstra a importância do uso de dados na área da saúde pública e reforça o potencial da ciência de dados como ferramenta de apoio à tomada de decisões.
